# 🚀 Cristea Mihai-Cătălin — Personal Portfolio

> **Live site:** [m1h41a.github.io](https://m1h41a.github.io)

A personal portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean and fast code.

## ✨ Features

- **Neural network particle background** that reacts to mouse movement
- **Custom animated cursor** with a glowing ring follower
- **Typed text effect** cycling through roles and skills
- **3D card tilt** effect on hover
- **Animated skill progress bars** triggered on scroll
- **Scroll-reveal animations** for every section
- **Floating code window** in the hero section
- **Glassmorphism dark theme** (purple → cyan gradient palette)
- **Fully responsive** — works on mobile, tablet, and desktop

## 📄 Sections

- **Hero** — Introduction with animated typing and social links
- **About** — Profile, contact details, and animated counters
- **Skills** — Python, C++, Mobile, SQL, C#, Java, Web, ML/AI
- **Projects** — Finance Dashboard, Quiz Site, Snake Game, University Projects
- **Experience** — Timeline of internships, education, and volunteering
- **Contact** — Email, LinkedIn, GitHub, Phone

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5 | Structure & semantics |
| CSS3 | Styling, animations, glassmorphism |
| JavaScript (ES6+) | Particle system, interactivity, effects |
| Google Fonts | Inter + Fira Code typography |
